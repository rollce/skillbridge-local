# SkillBridge Local

Multi-page volunteering match platform for real community coordination problems.

## Problem
People want to help locally, but NGOs struggle to match tasks by skills and availability.

## Solution
SkillBridge offers:
- opportunity board,
- skill/time-based recommendation engine,
- impact dashboard for measurable outcomes.

## Pages
- `/` Overview
- `/opportunities` Live opportunities
- `/matching` Match engine
- `/impact` KPI and outcomes

## API
- `GET /api/opportunities`
- `POST /api/match`

## UI Blocks
Built with **Chakra UI** (`Buttons`, `Cards/Boxes`, `Badges`, `Inputs`, `SimpleGrid`, `Alert`).
